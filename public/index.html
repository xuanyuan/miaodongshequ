编程技巧实例

arrayMax 返回数组中的最大值。
arrayMin 返回数组中的最小值。

将Math.max()与扩展运算符 (...) 结合使用以获取数组中的最大值。
将Math.min()与扩展运算符 (...) 结合使用以获取数组中的最小值。

const arrayMax = arr => Math.max(...arr); // arrayMax([10, 1, 5]) -> 10
const arrayMin = arr => Math.min(...arr); // arrayMin([10, 1, 5]) -> 1
